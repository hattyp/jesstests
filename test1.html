Let's build a navigation for this site so we can switch between pages.

  1. Build an unordered list in a section on the page below with five list items inside it.
  2. Add a link inside each list item. The links we need are:
    • Home - linking to home.html
    • About - linking to about.html
    • Icons - linking to gallery.html
    • Styleguide - linking to styleguide.html
    • Gallery - linking to gallery.html
  3. For each link, create the corresponding .html files in the root of this folder by duplicating the template.html file and renaming it.
  4. Click through each link to check that they work.
  5. Styling:Remove the bullet points and the padding on the left hand side of the list so it's not indented.
  6. Style the list items so they site horizontally across the page rather than vertically.
  7. Give each link inside the list items padding of 15px at the top and bottom and 20px at the left and right.
  8. Give each link inside the list items a background colour of rebeccapurple and make the text colour white.
  9. Give each link inside the list items a hover state so that the background colour changes when you hover the mouse over it.
