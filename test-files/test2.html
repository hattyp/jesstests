Test 2: Page structure

So for this one we're going to adjust the structure of the page a bit so it's a bit more semantically clear, and so that we can add a footer that sticks to the bottom of the site no matter how tall the page is.

Probably the best approach for this is to focus on setting it up on one page first and then copying and pasting it across to the others when you're happy with it. Let's do it in home.html first.

  1. Open the home.html file. Inside the body tags add a div with an id of 'container'. We can use an ID here rather than a class to target this with CSS because there'll only ever be one 'container' element on each page.
  2. Inside this 'container' div add opening and closing header tags. Inside these header tags add opening and closing nav tags.
  3. Inside these nav tags, cut and paste the ul you made in test1, and delete the section that it used to be contained in.
  4. By this point inside the body you should have a div with an ID of 'container', with a header inside of that, with a nav inside of that, with the ul you made last week inside of that! Bit like inception.
  5. Save this and check it on the page. It should be looking correct if the navigation you made last week is now the full width of the screen and there's a little gap at the top. This gap is here because by the ul has a margin by default. Reset this by setting it to 0 on the ul
  6. Neeeeext add an opening and closing main tag inside the 'container' div, but outside and below the header you just made.
  7. Neeeeext add an opening and closing footer tag inside the 'container' div, but outside and below the main you just made.
  8. If this is massively confusing, this article might be helpful, however if you're pretty happy with what we've done so far you can check it's correct by looking at this.
  9. So now you have a basic structure for your pages with a header, nav, main and footer element. Let's make this a bit clearer now. Add an h1 into the main element that just says 'Main' so we can see where the main section is, and add a p into the footer that says 'footer'.
  10. In your CSS, change the background colour of the footer to be a solid colour (omg I love the one you've used for the hover states in the navigation! where'd you get it?) and a cheeky bit of padding just so it's clear where the footer is AT.
  11. You should be able to see now the footer is sitting the full-width of the page but also just slots up underneath the main element. Next we're going to make him STAY at the bottom.
