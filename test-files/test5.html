Back in days of yore before Flexbox we had to use either tables or floated elements to organise elements around each other on a page. Flexbox has made all of this a lot easier but it takes some getting used to, so we're going to do some practice 'ere.

  1. In your 'home.html' file first add a 'banner' class to the first section inside the main element. We're gonna turn this into a banner image to bring the site to LIFE!
  2. Let's give this first 'banner' section a background image. You can find some free imagery here: https://unsplash.com/. I'd recommend choosing an image that is wider than it is tall given that it'll stretch the full width of the page. This sort of thing, perhaps?
  3. Next click on the image and then click the 'download free' button. Then move this downloaded image into the 'img' folder within the 'Jesstests' folder for this site. That's where we gonn' keep all the images. In your CSS file set the background image for the 'banner' section to be this downloaded image.
  4. Also remove the margins on this section, give it a minimum height of 300px and set the 'background-size' property to 'cover' so that the image sits full-width at the top of the page.
  5. Plz next copy the entirety of the main element you did last time (so including the sections inside it, the article and aside etc) from your 'home.html' into every page on the site. We gonna style them all at the same time.
  6. The main element will become our main flex container, so in the stylesheet make this the parent by making the display property 'flex'. Save this and refresh the page and you'll see the two 'section' elements are now side by side rather than stacked down the page. This is because the default 'flex-direction' style is 'row' - so unless you specify otherwise the elements inside a 'flex' container will appear left-to-right across the page. We want them to appear top-to-bottom down the page so also in the main element styles plz change the 'flex-direction' to be 'column' (so the elements inside it are stacked vertically rather than horizontally)
  7. We also want to control the positioning of the elements inside our section elements, so let's make these flex containers too by giving them the 'flex' display property. You can then change the 'margin' of the h1 tag to 'auto' to have the h1 text appear in the middle of the banner at the top too if you'd like.
  8. With the second of our section elements however we want the article and the aside to sit side by side, so we wont bother changing the 'flex direction' property, and instead we'll give the article a 'flex-grow' property of 3 and the aside a 'flex-grow' property of '1'
  9. We should now have the page looking a bit more websitey with a banner (complete with its own background image) full-width across the top, an article containing a bit of text on the left and an aside sitting on the right of the screen acting like a sidebar might.
